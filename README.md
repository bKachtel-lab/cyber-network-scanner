#  Cyber Network Scanner

##  Présentation
Cyber Network Scanner est un outil d'analyse de sécurité réseau développé en Python.

Il permet d'automatiser la phase de reconnaissance en cybersécurité en détectant :
- Les ports ouverts
- Les services actifs
- Les potentielles zones de vulnérabilités

Ce projet a été conçu dans un objectif pédagogique pour l'apprentissage des bases du pentesting et de l'analyse réseau.

---

##  Objectifs du projet

- Automatiser le scan réseau
- Identifier les services exposés
- Générer un rapport de sécurité
- Comprendre les bases de l'analyse SOC

---

##  Technologies utilisées

- Python 3
- Nmap
- Subprocess Python Module

Outils de cybersécurité :
- Wireshark (analyse réseau)
- VirtualBox (lab de test)
